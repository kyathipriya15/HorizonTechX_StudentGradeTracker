# 🎓 Student Grade Tracker

## 📖 Description
A Java console-based application developed for the Horizon TechX Java Internship.

## ✨ Features
- Add Student
- Display Students
- Calculate Average Marks
- Find Highest Marks
- Find Lowest Marks
- Summary Report

## 🛠 Technologies Used
- Java
- IntelliJ IDEA
- Object-Oriented Programming (OOP)
- ArrayList

## ▶ How to Run
1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Run `Main.java`.
